# 项目发展计划（精简版）

- In-Order 与 OOO 双实现
- 基础调试与测试链路（gtest + riscv-tests）

## 中期目标（3–6 个月）
- A 扩展与 C 扩展补齐
- 性能统计接口与分析能力
- 基础 CSR 与异常框架整理

## 长期目标（6 个月以上）
- MMU/分页与外设模拟原型
- 缓存与更复杂微架构探索
- 更完善的调试与可视化工具
