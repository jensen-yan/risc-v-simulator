# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the label strings to use if GitHub Issues are used for this repo.

Do not create these labels proactively. Apply or create labels only when the user explicitly asks to use GitHub Issues.

| Label in mattpocock/skills | Label in our tracker | Meaning                                  |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | Maintainer needs to evaluate this issue  |
| `needs-info`               | `needs-info`         | Waiting on reporter for more information |
| `ready-for-agent`          | `ready-for-agent`    | Fully specified, ready for an AFK agent  |
| `ready-for-human`          | `ready-for-human`    | Requires human implementation            |
| `wontfix`                  | `wontfix`            | Will not be actioned                     |

For local-only work, treat these as conceptual states rather than required labels.
